This project comprises of 5 source files

1. Library_search.java
2. CheckOut.java
3. CheckIn.java
4. BorrowerAdd.java
5. Fines.java

The files can be compiled in any order. 
Compilation can be done by 'javac ProgramName' . For example -'javac Library_search.java'
Thereafter ensure that the mysql connector exists in the folder and run the application 
by calling 'java Library_search'
